# Getting Started 

### `Docker compose up`


## Available Scripts

### `docker images`
see docker images

### `docker ps` 
see running containers

### `docker ps -a`
see all containers

### `docker rm <container name>`
remove container

### `docker rmi <image name>`
remove image

# Ports
web-app: 3000:3000

web-api: 8080:8080
